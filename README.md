# Black-Shot
a Tool to create some Viruses to Exploit the Windows system

## Requirements
* Linux
* Wine
* colorama
* Python3
* requests
* pibyone

## Installation

```bash
sudo apt-get install wine
sudo apt-get install python3
pip3 install pibyone
pip3 install requests
pip3 install  colorama
```
<h3>OR:</h3>


```bash
cd install/
python3 install.py
```
